# Service Overview

Mocha Messenger is only one part of the broader Mocha stack.

The public-facing service map currently looks like this:

- **Mocha Messenger**
  The Windows desktop client.
- **Mocha Passport**
  Identity, account, and supporting web services.
- **Notification Server**
  Presence, sign-in coordination, and conversation routing.
- **Switchboard**
  Conversation relay for chat and current media lanes.

For protocol-level details, the public `mochachat/Protocol` repository is the best place to start.
