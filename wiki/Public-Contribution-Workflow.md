# Public Contribution Workflow

Mocha Messenger is developed privately, but public collaboration is still welcome.

Workflow:

1. Open an issue, discussion, or public PR.
2. The Mocha team reviews it for fit, scope, and safety.
3. If accepted, the change may be manually ported into the private development repository.
4. The public thread is updated when the result is shipped or otherwise resolved.

This allows public collaboration without exposing the full production source tree.
