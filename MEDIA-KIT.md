# Media Kit

This page is the public-facing home for screenshots, logos, and basic project
description material that can be reused when talking about Mocha Messenger.

## What belongs here

- current screenshots of the client
- approved logos or branding assets
- short project descriptions for articles or directory pages
- release-channel screenshots when they help explain the current state of the
  client

## Suggested screenshot set

The public Messenger repo should ideally include:

- main buddy list window
- one-to-one conversation window
- voice and video conversation view
- Passport or service-status integration examples where helpful
- at least one screenshot showing the client in a classic Windows theme

## Suggested folder structure

If you want to publish assets inside the public repo, a clean structure would
be:

```text
media/
  screenshots/
  logos/
  badges/
```

## Short description

Mocha Messenger is the flagship Windows desktop client for the Mocha network,
focused on rebuilding the classic Messenger experience with a modern Mocha
service stack behind it.
