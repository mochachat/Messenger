# Mocha Messenger Roadmap

This page is a public high-level roadmap, not a promise list.

Mocha Messenger is being rebuilt in stages so the classic Messenger experience can sit on top of the Mocha network cleanly.

## What already exists

- Passport-backed sign-in
- buddy list and categories
- presence and personal messages
- one-to-one and group conversations
- file transfer
- current voice and video calling support
- service-status integration
- in-client updating

## Current near-term polish areas

- more call and media polish
- compatibility refinement across Windows versions and themes
- continued UI/UX cleanup
- broader private-beta feedback integration

## Future-facing areas

- deeper Passport-backed web integration
- future public/community documentation expansion
- future add-in ecosystem planning
- broader companion-service integrations where they fit the Messenger experience

## About future add-ins

Mocha is interested in an add-in model, but the goal is to make it useful and safe rather than shipping a rushed plugin surface.

Current public expectation:

- add-ins are a future-facing area, not a broadly released end-user feature today
- public design discussion is welcome
- compatibility, trust, and abuse resistance matter as much as flexibility
