# Support and Feedback

This repository is one of the main public intake points for Mocha Messenger feedback.

## Best places to report things

- product or UI feedback: public issue or discussion
- protocol documentation question: `mochachat/Protocol`
- service health question: [https://mochachat.net/support/status.html](https://mochachat.net/support/status.html)
- download/update questions: [https://mochachat.net/download/download.html](https://mochachat.net/download/download.html)

## What makes a bug report useful

Please include:

- exact build number
- Windows version
- whether custom shell or theme modifications are in use
- reproduction steps
- whether the issue happens every time
- screenshots if possible

## What makes a feature request useful

Please explain:

- what problem it solves
- what you expected to happen
- whether there is MSN Messenger precedent, if relevant
- why it fits Mocha specifically

## A note on the private codebase

The shipped client and services are still developed in a private repository. Public reports here are still valuable, but accepted changes may be implemented privately and then reflected back into public docs or release notes later.
