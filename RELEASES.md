# Public Release Notes

This page is the public-facing release index for Mocha Messenger.

It should stay shorter and more audience-friendly than the private internal
release notes used for packaging and updater workflows.

## What to publish here

- major or meaningful public dev/beta milestones
- high-level feature additions
- notable bug fixes
- compatibility improvements worth calling out publicly

## What not to publish here

- private infrastructure details
- sensitive moderation or admin changes
- deployment internals
- anything that would reveal more of the private implementation than intended

## Suggested structure

For each public release note entry, include:

- version
- channel
- release date
- short summary
- top fixes or additions

## Example entry shape

```text
## DEV 1.1.xxxx

- Improved conversation polish and update behavior
- Added new public docs and support guidance
- Fixed several Windows compatibility and theme issues
```
